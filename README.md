# 세움 홈플래너 (SEUM Home Planner)

영업팀이 고객 상담 시 **3D 도면**으로 집을 설명하고, 그 자리에서 **방을 추가/확장하거나 가구를 배치**해볼 수 있는 웹 기반 주택 설계 도구입니다. (한샘 홈플래너 참고)

세움은 집을 짓는 회사이므로, 방 도면을 3D로 변환한 뒤 **방 추가 · 넓히기/좁히기 · 다락 · 다용도실** 등을 쉽게 바꿀 수 있도록 만들었습니다.

---

## 실행 방법

별도 설치(빌드)가 필요 없습니다. Node.js만 있으면 됩니다.

```bash
npm start
# 또는
node server.js
```

브라우저에서 **http://localhost:5173** 접속.

> 3D 렌더링(Three.js)은 CDN에서 불러오므로 최초 실행 시 인터넷 연결이 필요합니다.
> 사내망 등 폐쇄망에서 쓰려면 `three` 라이브러리를 받아 `index.html`의 importmap 경로만 로컬로 바꾸면 됩니다.

## 주요 기능

### 1. 2D 평면 편집 (집짓기 도면)
- **집 외곽(외벽) 먼저 그리기**: 좌측 **🏠 집 외곽(외벽) 그리기** → 모서리를 **클릭해 직선 구간으로 이어** 그림(사각형뿐 아니라 **ㄱ자·ㄷ자 등 자유로운 형태**). 그릴 때 **각 변 길이(치수)가 실시간 표시**됨. **더블클릭=그만 그리기(닫지 않아도 됨)**, 시작점 클릭=닫기, Esc=취소. 직각 스냅으로 반듯하게. 3D에서 실제 외벽·지붕·바닥으로 표현
- **공간 추가**: 좌측 팔레트에서 **방·화장실·다용도실·현관·드레스룸·데크·포치·썬룸**(세움 표준 공간) 클릭(추가) 또는 **도면으로 드래그**. 닫힌 외벽 안으로 드래그하면 **그 안에 맞춰 자동 배치**. (데크·포치는 개방형 — 3D에서 바닥만)
- **자동 정렬(스냅)**: 방을 드래그하면 외곽·옆방 모서리에 자동으로 달라붙어 빈틈 없이 배치
- **방 그리기**: **✏️ 방 그리기** 로 외곽 안에서 드래그해 방을 직접 그림
- **넓히기 / 좁히기**: 방을 선택하면 나타나는 8개 핸들을 드래그 (100mm 단위 스냅)
- **이동**: 방을 드래그
- **정밀 수정**: 우측 속성 패널에서 가로/세로/위치를 mm 단위로 직접 입력
- **면적 자동 계산**: m² 및 평수 표시

### 2. 3D 입체 보기 (고객 상담용)
- 상단 **[3D 입체]** 버튼으로 전환 → 2D 도면이 실시간으로 입체 벽·바닥으로 변환
- 마우스 드래그로 회전, 휠로 줌
- **입체 / 평면 / 정면** 카메라 프리셋, **천장 보기** 토글
- **다락**은 경사 지붕으로 표현
- **개방형 벽(트기/막기)**: 두 가지 방법 — ① 좌측 *공간 추가*의 **🧱 벽 트기/막기** 토글을 켜고 2D에서 **벽을 클릭**(맞닿은 두 방이 함께 처리돼 통로가 바로 뚫림, 트인 벽은 점선 표시) ② 방 선택 → 속성 패널에서 북/남/동/서 벽 버튼. 거실·주방·LDK 같은 트인 공간이나 막힌 통로를 자유롭게 조절
- 📷 **캡처**로 3D 화면을 이미지로 저장해 고객에게 전달

### 3. 외장재 · 지붕 · 창호 (집짓기 특화)
- **외장재**: 메탈사이딩 · 시멘트사이딩 · 세라믹사이딩 · 스타코 · 벽돌 · 목재 · 석재 선택, **색상 자유 변경**(팔레트 + 컬러 피커)
- **사실적 질감**: 자재를 고르면 3D에서 단색이 아니라 **실제 결·줄눈·패턴**(벽돌 줄눈, 사이딩 골, 목재 결, 석재, 지붕 슁글, 바닥 마루/타일, 가구 패브릭/원목)으로 표현 — 선택한 색상이 질감 위에 입혀짐 (외부 이미지 불필요, 폐쇄망 동작)
- **지붕**: 평지붕 · **박공지붕** · **비대칭 박공** · 우진각(모임) · 외쪽지붕, 색상 변경
- **창호**: **2중창 · 미닫이창 · 여닫이창(1/2짝) · 고정창 · 폴딩도어 · 폴딩창 · 발코니 대형창 · 현관문** — 좌측 `창호` 탭에서 방 가장자리(벽)로 드래그해 바로 배치
- **실제로 뚫린 개구부**: 창/문을 놓으면 3D에서 그 자리의 벽이 실제로 뚫려(내벽·외장 모두) 유리 너머가 보임
- 창은 벽을 따라 드래그로 이동, 폭/높이/하단높이/위치/창틀색상 개별 조정 (속성 패널에서 mm 단위)
- 외장 마감은 별도 테두리 박스가 아니라 **건물 외곽에 면한 실제 벽체**에 입혀짐 (안쪽 공유 벽은 제외)
- 외장재·색상을 고르면 3D 외관이 자동으로 켜지고, 우측 속성 패널 또는 3D 우측 상단 **외장재 / 지붕** 버튼으로 ON/OFF 가능
- 외장 색상은 빠른 선택 **팔레트(계열별 30여 종)** + 컬러 피커로 자유 지정

### 4. 가구 · 가전 라이브러리
- 가구 / 가전 / 욕실 / 소품 카테고리
- 가전에 **AV·스크린골프**(빔프로젝터·스크린·스피커·노래방 기계·스크린골프 기계) 포함 — 골프존·룸카페 상담용
- 카드를 도면으로 **드래그**하거나 **더블클릭**으로 배치
- 회전(15° 단위 핸들 또는 90° 버튼), 복제, 삭제
- 2D에 배치한 가구가 3D에도 그대로 입체로 나타남

### 5. 기본 도면 라이브러리 (세움 제품)
- 상단 **📐 기본 도면** 버튼 → 세움 판매 제품의 기본 도면을 불러와 **바로 수정**
- **제품 종류별 분류**: 🏠 주택 · 🏕️ 체류형 쉼터 · 🛖 농막 — 상단 칩으로 필터
- 내장 기본 도면: 단독주택 99㎡, **농막 20㎡(6평)**, **체류형 쉼터 33㎡(10평)**, 킨텍스 전시모델 10평
- **내 기본 도면 추가(클라우드 불필요)**: 📐 기본 도면 창에서 이 기기의 *내 기본 도면* 라이브러리에 등록 → 다음부터 불러와 수정만. 카드의 ✕로 삭제
  - **"➕ 현재 도면을 기본 도면으로 추가"**: 지금 작업 중인 도면을 등록
  - **"📁 파일 등록"**: 기존에 가진 파일을 한 번에 여러 개 골라 등록 — **`.json`**(편집 도면)은 그대로, **PDF·이미지**(PNG/JPG 등)는 *밑그림 배경 도면*으로 등록(불러오면 배경으로 깔려 따라 그리기). **CAD(.dwg/.dxf)** 는 브라우저에서 못 열어 PDF/이미지로 내보내 등록하라고 안내
- (클라우드 설정 시) **영업팀 공용 기본 도면** 갤러리도 함께 표시
- **팀 공용으로 저장**: ☁ 클라우드 → "📐 기본 도면으로 저장"으로 현재 도면을 제품 종류와 함께 팀 공용 라이브러리에 등록 → 다음부터 불러와 수정만 하면 됨

### 5-1. 밑그림 (PDF·이미지·DXF 도면 따라 그리기)
- 상단 **🖼️ 밑그림** 버튼 → 가지고 있는 **PDF / PNG / JPG / DXF(캐드)** 도면을 편집 화면 배경에 깔기
- **DXF(캐드)**: 캐디안·오토캐드에서 *DXF로 저장*해 올리면 선·도형을 읽어 배경으로 표시(좌표 크기 자동 인식). `.dwg`는 캐드에서 DXF로 저장 후 사용
- **축척 맞추기**: 도면에서 길이를 아는 두 점을 클릭하고 실제 mm를 입력 → 밑그림이 실제 크기로 보정됨
- **✏️ 도면 위에 방 그리기**: 좌측 *공간 추가*의 토글을 켜고, 방 종류를 고른 뒤 밑그림 위에서 **대각선으로 드래그**하면 그 자리에 방이 생성됨 → 빠르게 따라 그려 **2D·3D 편집 도면**으로 변환 (3D는 자동 생성). 다 그리면 밑그림 제거 후 "기본 도면으로 저장"
- ※ PDF/이미지를 *완전 자동*으로 인식해 방·벽으로 바꾸지는 않습니다(도면마다 표기가 달라 정확도 낮음). 위 드래그 방식으로 빠르게 따라 그리는 방식입니다.
- 투명도 조절·숨기기 가능. (PDF 변환은 최초 1회 인터넷 연결 필요 — pdf.js CDN)
- ※ PDF는 "그림"이라 *가져오기(.json)* 로는 열 수 없습니다. 반드시 **밑그림**으로 불러오세요.

### 6. 클라우드 저장 · 공유 (Supabase)
- 상단 **☁ 클라우드** 버튼 → 영업사원 계정 **로그인**
- **클라우드 저장**: 도면을 서버에 저장 (기기 바뀌어도 유지)
- **영업사원 간 공유**: "공유" 켜면 동료의 *공유 도면* 목록에 노출
- **고객용 공유 링크**: `?id=...` 링크 복사 → 고객이 로그인 없이 도면 열람
- 설정 방법: [`supabase/README.md`](./supabase/README.md) (설정 전에도 나머지 기능은 정상 동작)

### 7. 인쇄 / 저장 / 공유
- **🖨 인쇄**: 평면도 + 3D 투시도 + **공간 면적표** + **외장/지붕/창호 사양**을 A4 견적 도면 한 장으로 출력 (PDF 저장 가능)
- **📷 캡처**: 현재 2D 도면 또는 3D 화면을 PNG 이미지로 저장
- **저장 / 불러오기**: 브라우저에 도면 보관 (이름별)
- **내보내기 / 가져오기**: `.json` 파일로 백업·전달
- 실행 취소/다시 실행 (Ctrl+Z / Ctrl+Y), 자동 임시 저장

## 프로젝트 구조
```
index.html          화면 구성 + Three.js importmap
server.js           의존성 없는 정적 서버
styles/app.css      스타일
config.js           런타임 설정 (Supabase URL/anon key)
supabase/           DB 스키마(schema.sql) + 클라우드 설정 가이드
src/
  data.js           공간 종류 · 가구 라이브러리 · 기본 도면
  store.js          상태/되돌리기/저장 관리
  editor2d.js       2D 평면 편집기 (Canvas)
  viewer3d.js       3D 뷰어 (Three.js)
  templates.js      기본 도면 라이브러리 (주택/체류형 쉼터/농막)
  cloud.js          Supabase 클라우드 연동 (로그인/저장/공유)
  ui.js             툴바·라이브러리·속성 패널·모달
  main.js           진입점
```
